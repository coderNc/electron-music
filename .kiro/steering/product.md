# Local Music Player

A cross-platform desktop music player built with Electron for playing local audio files.

## Core Features

- Local folder scanning and music library management
- Audio playback with full controls (play, pause, skip, seek, volume)
- Metadata parsing from audio files (title, artist, album, cover art)
- Playlist and queue management
- Multiple playback modes (sequential, shuffle, repeat-one, repeat-all)
- Search and filtering
- Light/dark theme support
- Keyboard shortcuts
- Data persistence across sessions

## Supported Audio Formats

MP3, FLAC, WAV, AAC, OGG, M4A, WMA, AIFF, ALAC, OPUS
